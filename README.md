# 📱 M-Box Login UI

A clean and modern Flutter login screen UI built for learning Flutter layouts and widget composition. This project focuses on creating an attractive authentication interface using Flutter's built-in widgets, custom fonts, and image assets.

> **Note:** This project is a UI design only. It does not include authentication, backend integration, or API functionality.

---

## ✨ Features

- Modern and clean login screen UI
- Custom logo image
- Custom Rubik fonts
- Email input field
- Password input field
- Styled "Forgot Password?" text
- Styled "Log In" button
- Sign Up prompt
- Responsive layout using Flutter widgets
- Built entirely with Flutter Material components

> **Note:** This project is a static UI design. The login button, "Forgot Password?" text, and "Sign Up" prompt are for interface demonstration only and do not include any functionality.

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio or VS Code
- Android Emulator or Physical Device

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/m_box_login_ui.git
```

2. Navigate to the project folder

```bash
cd m_box_login_ui
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
